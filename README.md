# generator-dadamp

> Vue + Vuex + Vue Router + Vue Impression + Webpack 3

[English Version](./README_EN.md)

## 安装

首先, 安装`Yeoman`和`generator-dadamp`:

```bash
yarn global add yo generator-dadamp
```

## 创建新项目

```bash
yo dadamp
```

## 升级旧项目

```bash
cd YOUR_PROJECT_FOLDER
yo dadamp --upgrade
```
